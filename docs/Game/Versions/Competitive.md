The competitive study version of the Coop Game, designed to explore competitive dynamics and decision-making under competitive pressure.

## Overview

This version explores competitive dynamics by introducing competitive elements into the game structure. It is designed for a specific study examining how competitive pressure affects children's cooperative behavior and decision-making.

## Levels

The competitive version uses a curated level set:
- **Level 0**: Tutorial (standard introduction)
- **Levels 17-19**: Competitive gameplay scenarios
- **Levels 21-27**: Advanced competitive challenges

## User IDs

Access is restricted to specific study participants:
- User ID: 6224
- User ID: 7614

This version is designed for these specific participants and their study cohort.

## Features

- Competitive game mechanics and scenarios
- Scoring systems that emphasize competition
- Dynamic difficulty adjustment
- Specialized feedback for competitive gameplay

## Study Information

This competitive variant tracks:
- Participant choices under competitive conditions
- Willingness to cooperate vs. compete
- Performance in competitive scenarios
- Behavioral patterns unique to this version

## Deployment

This version is deployed from the `prod-competitive` branch to Google Cloud Run (Gen 2).
