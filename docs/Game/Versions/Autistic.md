## Overview

This version is optimized for autistic participants, with design choices and gameplay mechanics tailored to their needs and preferences. It is part of the research study exploring how different game experiences affect cooperative behavior in autistic children.

## Levels

The autistic version includes levels 0-10, covering the full game progression adapted for this participant group.

## Features

- UI/UX optimized for autistic participants
- Adapted game mechanics and pacing
- Customized feedback and reward systems
- Sensory-friendly design considerations

## Study Information

This version is used in the autistic study cohort and tracks:
- Participant choices and cooperation patterns
- Performance metrics specific to this variant
- Behavioral differences compared to other versions

## Deployment

This version is deployed from the `prod-Autistic-exp` branch to Google Cloud Run (Gen 2).
