## Overview

The main game version runs the core Coop Game experience and supports multiple language variants:
- Deaf version
- Macedonian version
- FullTFT version

This version is the primary deployment for most participants and research studies.

## Levels

The main game includes levels 0-10, covering:
- **Level 0**: Tutorial/Introduction
- **Levels 1-10**: Core gameplay progression

## Features

- Multi-language support
- Accessibility configurations
- Full game flow from introduction through completion
- Session tracking and data collection
- Integration with user management and analytics systems

## Deployment

This version is deployed to Google Cloud Run (Gen 2) with automatic deployment from the `main` branch.
